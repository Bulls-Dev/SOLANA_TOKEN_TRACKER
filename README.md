# ⚡ OXBLOCK TRACKER - Solana Raydium Monitor

[![Live Demo](https://img.shields.io/badge/Demo-Live-green?style=for-the-badge)](https://solana-token-tracker-lake.vercel.app/)
[![Source Code](https://img.shields.io/badge/Source-GitHub-white?style=for-the-badge&logo=github)](https://github.com/Bulls-Dev/SOLANA_TOKEN_TRACKER.git)

**OXBLOCK TRACKER** is a high-performance, real-time monitoring engine designed for the Solana ecosystem. It provides a reliable feed of freshly listed tokens on **Raydium** by parsing on-chain logs with millisecond precision.

Designed for the "Solana Trenches", this tool serves as the perfect front-end or data-source for sniper bots and advanced trading strategies.

---

## 🚀 Key Features

* **Real-time On-chain Monitoring**: Directly parses Solana mainnet logs to detect new liquidity pools instantly.
* **Deep Metadata Extraction**: Automatically saves and displays critical information:
    * Token Creator address.
    * Initial liquidity & balances.
    * Transaction signatures (SIG) for immediate verification.
* **Optimized for Speed**: Built to handle the high throughput of Raydium AMM & CPMM programs.
* **Developer Friendly**: Open-source architecture, perfect for integration into larger trading systems.

## 🛠 Tech Stack

* **Network**: Solana Mainnet-Beta
* **DEX**: Raydium (AMM & CPMM)
* **Frontend**: React / Next.js (Tailwind CSS for the dark-mode "Trench" UI)
* **Data**: On-chain RPC Stream / WebSocket

## 📥 Getting Started

### Prerequisites
- Node.js (v18+)
- A Solana RPC Provider (Helius, QuickNode, or Shyft recommended for speed)
